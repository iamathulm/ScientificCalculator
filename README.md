# Scientific Calculator (Python + Tkinter)

A beginner-friendly scientific calculator built using **Python** with a **Tkinter GUI**.

## Features
- Basic arithmetic: `+  -  *  /  %`
- Powers: `^` and `**`
- Roots: `sqrt(x)` and `root(x, n)`
- Trigonometry: `sin(x), cos(x), tan(x)` (DEG/RAD mode)
- Logarithms: `ln(x)`, `log(x)`, `log(x, base)`
- Constants: `pi`, `e`
